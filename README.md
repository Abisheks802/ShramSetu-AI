# ShramSetu AI

ShramSetu AI is a multilingual AI-powered assistant designed to help Indian workers understand labor laws, ESIC, PF, gratuity benefits, and complaint mechanisms through conversational AI.

## Key Features
- Multilingual support
- Voice-enabled interface
- Eligibility calculators
- Complaint drafting support
- AI-powered compliance education

## Tech Stack
- AWS Cloud
- Python backend
- NLP + LLM-based assistant
- WhatsApp chatbot integration

## Vision
To bridge the gap between policy and people using AI for Bharat.
